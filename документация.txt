1) Данные подключения хранятся в файле db.php
2) На главной странице внизу форма регистрации 
3) Из пункта меню можно попасть на форму авторизации
4) После авторизации можно получить accsses_token для работы с api 
5) /api/get_user.php получаем json ответ с id пользователя, адресом и ФИО
6) /api/edit_user.php отправляем api запрос на изменение адреса и фио
